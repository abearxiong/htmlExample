文字的三种渐变

1. 使用background-cli和text-fill-color的方法
渐变的颜色

a. backgound-image属性

图片的背景，url(图片)，获取可以填充的界面

非继承，默认值none。背景占据元素的全部尺寸，包括内边距和边框，但不包括外边距。

b. -webkit-background-clip: text

规定背景的绘制区域，默认值border-box,继承性no，JavaScript语法object.style.backgroundClip="content-box"

border-box 背景被裁剪到边框盒|padding-box 背景被裁剪到内边距框 | content-box 背景被裁剪到内容框。

-webkit-background-clip: text支持chrome，是对区块内的文字进行裁剪。文字的背景为区块的背景，文字以外的区域都将被裁剪掉。

c. -webkit-text-fill-color:transparent对文本的颜色设置透明

设置文本的填充颜色，color: transparent也可。mdn不推荐使用

c. -webkit-linear-gradient(bottom,red, #fd8403,yellow)线性变换颜色

ilnear-gradient()函数是用户创建一个线性渐变的图像

起点颜色，和终点颜色，创建的更多，渐变色更加的复杂

第一个参数可以设置一个角度deg，设置线性渐变的一个过程。

也可以设置bottom等，方向渐变的路径，-webkit和普通的不一样，普通的有to bottom. 

.gradient-text{
  text-align: center;
  text-indent: 30px;
  line-height: 50px;
  font-size: 40px;
  font-weight: bolder;
  position: relative;
}
.one{  
  background-image:-webkit-linear-gradient(bottom,red,#fd8403,yellow); 
  -webkit-background-clip: text; 
  -webkit-text-fill-color:transparent; 
}
2. 使用mask-iamge
渐变的颜色

a. -webkit-mask-image

图片蒙板的效果，对图片进行遮罩。

b. two[data-content]:after{},可以设置参数，html的内容可以绑定到css样式当中。 

.two[data-content]::after {
  content:attr(data-content);
  display: block;
  position: absolute;
  color: yellow;
  left: 0px;
  top: 0px;
  z-index: 2;
  -webkit-mask-image: -webkit-gradient(linear,0 0,0 bottom, from(yellow), to(rgba(0,0,255,0)));
}
